# Handspeak (static demo)

Triển khai GitHub Pages:
1. Tạo repo (ví dụ: `Handspeak`) → **Settings → Pages** → Source: `main` / root.
2. Tải toàn bộ thư mục này lên (nếu dùng `/docs`, chuyển file vào `/docs`).
3. Mở site qua HTTPS (GitHub Pages mặc định HTTPS). Trình duyệt sẽ xin quyền camera/micro.
